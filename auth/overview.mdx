## Authentication in Nyra Wallet

Nyra Wallet provides secure authentication for your digital wallet needs. Our authentication system ensures that only authorized users can access their wallets and perform transactions.

## Security Features

- **Multi-factor authentication** for enhanced security
- **JWT tokens** for secure session management
- **Rate limiting** to prevent brute force attacks
- **Encrypted storage** of sensitive user data

## Getting Started

To begin using authentication in your application:

1. **Register a new user** using the registration endpoint
2. **Authenticate users** with the login endpoint
3. **Use the returned JWT token** for subsequent API calls

<Note>
  All authentication endpoints require HTTPS and follow OAuth 2.0 standards for security best practices.
</Note>
