## Business Management in Nyra Wallet

Nyra Wallet allows users to register and manage businesses, then create API clients for secure third-party integrations. This system provides a secure way to manage business data while enabling external applications to access specific business information.

## How It Works

<Steps>
<Step title="User Authentication">
  Users authenticate using JWT tokens to access business management features.
  
  <Check>
  JWT token provides full access to business operations.
  </Check>
</Step>

<Step title="Business Registration">
  Authenticated users can register new businesses with basic information.
  
  <Check>
  Each business gets a unique ID and is associated with the user.
  </Check>
</Step>

<Step title="API Client Creation">
  Users create API clients for their businesses to enable integrations.
  
  <Check>
  API clients get client ID and secret for authentication.
  </Check>
</Step>

<Step title="API Access">
  Third-party applications use client credentials to access business data.
  
  <Check>
  Limited scope access based on API client permissions.
  </Check>
</Step>
</Steps>

## API Client Operations

- **Create API Client**: Generate new client credentials for a business
- **Regenerate Secret**: Update client secret if compromised
- **View Client Details**: Monitor API client usage and status
- **Access Business Data**: Use client credentials to retrieve business information

## Security Features

- **JWT Authentication**: Secure user authentication for business management
- **Client Credentials**: Separate authentication for API clients
- **Business Ownership**: Users can only manage their own businesses
- **Scoped Access**: API clients have limited access to specific business data
