---

## Overview

Retrieves the balance of the primary wallet for the business associated with the authenticated API client.

## Authentication

- **Type**: API Client
- **Headers**:
  - `x-client-id`: The client ID (required, string)
  - `Authorization`: `Bearer <client_secret>` (required, string)

## Response

### Success Response (200 OK)

```json
{
  "success": true,
  "message": "Wallet balance fetched successfully",
  "data": {
    "businessId": "string",
    "businessName": "string",
    "balance": 0
  }
}
```

## Error Handling

- **401 Unauthorized**: Invalid client ID or secret
- **403 Forbidden**: API client not authorized for the business
- **404 Not Found**: Business or its wallet not found
