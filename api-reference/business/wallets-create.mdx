## Overview

Creates a new wallet for a customer under the business associated with the authenticated API client.

## Authentication

- **Type**: API Client
- **Headers**:
  - `x-client-id`: The client ID (required, string)
  - `Authorization`: `Bearer <client_secret>` (required, string)

## Request Body

```json
{
  "first_name": "string",
  "last_name": "string",
  "name_on_account": "string",
  "middlename": "string",
  "dob": "string",
  "gender": "string",
  "title": "string",
  "address_line_1": "string",
  "address_line_2": "string",
  "city": "string",
  "country": "string",
  "phone_number": "string",
  "email": "string",
  "bvn": "string"
}
```

### Required Fields

- `first_name`: Customer's first name (string)
- `last_name`: Customer's last name (string)
- `dob`: Date of birth in ISO format (string)
- `gender`: Customer's gender (string)
- `title`: Customer's title (string)
- `address_line_1`: Primary address line (string)
- `city`: City (string)
- `country`: Country (string)
- `phone_number`: Valid phone number (string)
- `email`: Valid email address (string)
- `bvn`: Bank Verification Number (11-digit string)

### Optional Fields

- `name_on_account`: Name on the account (string)
- `address_line_2`: Secondary address line (string)

## Response

### Success Response (201 Created)

```json
{
  "success": true,
  "message": "Wallet created successfully",
  "data": {
    "wallet_id": "string",
    "account_number": "string",
    "bank_name": "string",
    "owners_fullname": "string",
    "frozen": false,
    "business_id": "string",
    "isFloat": false,
    "created_at": "string",
    "updated_at": "string"
  }
}
```

## Error Handling

- **400 Bad Request**: Invalid request payload or BVN not an 11-digit string
- **401 Unauthorized**: Invalid client ID or secret
- **403 Forbidden**: API client not authorized for the business
