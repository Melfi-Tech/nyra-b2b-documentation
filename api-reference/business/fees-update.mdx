description: 'Updates business fee configuration for a specific business with partial updates support'
openapi: 'PUT /business-fees/{businessId}'
---

## Overview

Updates business fee configuration for a specific business. Only provided fee types will be updated, allowing for flexible fee management without affecting unchanged configurations.

## Authentication

- **Type**: JWT Token
- **Headers**:
  - `Authorization`: `Bearer <jwt_token>` (required, string)
- **Required Role**: ADMIN or SUPER_ADMIN

## Path Parameters

- **businessId**: The business ID to update fees for (required, string)

## Request Body

All fee configurations are optional. Only the fee types you want to update need to be included in the request.

## Response

Returns the updated fee configuration with all fee types and business details reflecting the changes made.

## Error Handling

- **400**: Invalid businessId
- **401**: Invalid JWT token
- **403**: User doesn't have ADMIN or SUPER_ADMIN role
- **404**: No business fees found for the business ID
