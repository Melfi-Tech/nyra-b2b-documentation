## Authentication

<Warning>
  This endpoint requires a valid JWT token for authentication.
</Warning>

- **Method**: JWT token in `Authorization: Bearer <token>` header
- **Scope**: Returns only businesses owned by the authenticated user
- **Permissions**: Full access to business data for owned businesses

## Response Details

The response includes an array of all businesses owned by the authenticated user, with each business containing:
- **ID**: Unique business identifier
- **Name**: Business name
- **Address**: Business address
- **Business Type**: Predefined business category
- **Creator**: User ID of the business owner
