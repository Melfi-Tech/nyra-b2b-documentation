## Overview

Retrieves details of the parent wallet for a business, accessible by an authenticated user.

## Authentication

- **Type**: JWT Token
- **Headers**:
  - `Authorization`: `Bearer <jwt_token>` (required, string)

## Path Parameters

- `businessId`: Business ID (required, string)

## Response

### Success Response (200 OK)

```json
{
  "success": true,
  "message": "Wallet retrieved successfully",
  "data": {
    "wallet_id": "wallet_54321",
    "account_number": "1234567890",
    "business": {
      "id": "business_12345",
      "name": "Business Inc"
    },
    "owner": {
      "user_id": "user_12345",
      "email": "user@example.com"
    },
    "frozen": false,
    "created_at": "2025-07-26T04:22:00.000Z",
    "updated_at": "2025-07-26T04:22:00.000Z"
  }
}
```

## Error Handling

- **400 Bad Request**: Invalid businessId
- **401 Unauthorized**: Invalid JWT token
- **404 Not Found**: Wallet or business not found
