## Overview

Retrieves list of available banks. This endpoint provides a comprehensive list of all supported banks for transfers.

## Authentication

- **Type**: API Client
- **Headers**:
  - `x-client-id`: Client ID (required, string)
  - `Authorization`: `Bearer <client_secret>` (required, string)

## Prerequisites

- API client must be authenticated and authorized for the business

## Response

Returns an array of bank objects with bank codes, names, and short names.

## Error Handling

- **401**: Invalid client credentials
